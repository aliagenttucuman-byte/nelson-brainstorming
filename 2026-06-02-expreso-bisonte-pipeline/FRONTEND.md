# FRONTEND — UX no técnica para Expreso Bisonte

## Objetivo UX
Que un usuario operativo pueda resolver cruces de datos sin saber joins, llaves ni transformaciones técnicas.

## Estructura de pantalla
1. **Carga de archivos**
   - Drag & drop + selector.
   - Lista de archivos con acciones rápidas.
2. **Objetivo en lenguaje natural**
   - Campo principal + ejemplos clickeables.
3. **Plan sugerido**
   - Pasos legibles, sin jerga técnica.
4. **Resultado**
   - Tabla preview + descarga.
5. **Biblioteca de procedimientos**
   - Usar, mover prioridad, duplicar, eliminar.

## Reglas UX clave
- Mostrar una sola acción principal por etapa.
- Errores en lenguaje humano (no stacktrace).
- Botones mobile full-width en acciones críticas.
- Scroll horizontal seguro para tablas grandes.

## Mejoras UX planificadas
- Botón explícito "Guardar prompt" (manual).
- Búsqueda por nombre de procedimiento.
- Badges de "más usado" / "reciente".
- Mini resumen automático tras cada ejecución.
