## ¿Qué es?

Directorio comunitario de **APIs públicas y gratuitas**, categorizado y mantenido por miles de contribuidores.

- Cubre: clima, finanzas, gobierno, música, deportes, mapas, ML, seguridad, transporte, salud, fotos, videos, criptos, y más
- Cada entrada indica: autenticación requerida, HTTPS, CORS
- No es una librería ni un tool — es una **referencia de recursos**
